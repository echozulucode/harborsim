# harborsim

central "harbor" for docking many simulators.

## Quick Start

### Development

```
# terminal 1
cd backend
uv run .\main.py
# http://localhost:8000/

# terminal 2
cd frontend
npm run dev
http://localhost:5173/
```

Optionally, you can go directly to http://localhost:8000/docs with just the backend running to test the API.

- Use the "Authorize" button in the upper-right corner first.
